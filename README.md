## What is EngageZ?

EngageZ is a gamified engagement marketplace on Solana where:
- **Brands** create missions with SOL/USDC reward pools
- **Creators** compete by producing content and get scored automatically
- **Engagers** earn by liking, commenting, and sharing the best content

All rewards distributed transparently. All scoring on-chain.

## How It Works

1. Brand deposits reward pool → treasury wallet (publicly visible)
2. Admin approves → mission goes live
3. Creators submit content → auto-scored by engagement
4. Winners receive SOL/USDC directly to their wallet
